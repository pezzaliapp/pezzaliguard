Benvenuto su PezzaliGuard

PezzaliGuard è il tuo database personale di numeri spam. I dati restano sul telefono, niente account, niente backend. Per iniziare ti consigliamo questi 3 passi:

    Importa la Lista italiana curata ~7 numeri pre-segnalati pronti da usare
  2. 2
    Aggiungi i tuoi numeri Quelli che ti hanno chiamato e vuoi ricordare
  3. 3
    Esporta in JSON Quando sei pronto per usarlo in un'app iOS

Privacy

Tutti i dati restano sul tuo dispositivo.

  • Nessun server, nessun cloud, nessun account.
  • Nessun tracking, nessun cookie pubblicitario.
  • Nessuna API esterna — la PWA funziona anche offline.
  • Storage usato: localStorage del browser.

Nota importante. Su iPhone, una PWA non può bloccare realmente le chiamate. Questa app gestisce il database e produce un file spam-numbers.json che potrà alimentare una futura app iOS nativa con CallKit / Call Directory Extension.
